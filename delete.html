모두의 복약 — 데이터 삭제 요청

「모두의 복약」 앱 이용자는 아래 방법으로 언제든지 데이터 삭제를 요청할 수 있습니다.

방법 1. 앱에서 직접 삭제

  1. 앱 하단의 더보기 탭을 엽니다.
  2. 개인정보 설정을 누릅니다.
  3. 모든 데이터 삭제를 누르고 확인합니다.

이 방법으로 기기에 저장된 앱 데이터(프로필 연결, 설정, 예약된 알람)가 즉시 삭제됩니다.

방법 2. 이메일로 서버 데이터 삭제 요청

newblueday26@gmail.com 으로 앱에 등록한 이름과 함께 삭제를 요청해 주세요. 요청 접수 후 7일 이내에 서버에 저장된 데이터를 삭제해 드립니다.

삭제되는 데이터와 보관 기간

  • 삭제 대상: 이름, 성별, 생년월일, 복약 일정, 복용 기록 — 요청 시 전부 삭제됩니다.
  • 음성 데이터: 음성은 인식 처리 후 당사가 별도로 보관하지 않으므로 삭제할 저장분이 없습니다.
  • 추가 보관: 법령상 보관 의무가 있는 경우를 제외하고, 삭제 후 별도의 보관 기간 없이 파기됩니다.

자세한 내용은 개인정보처리방침을 참고해 주세요.
